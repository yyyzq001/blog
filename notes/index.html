    一、mac上新建git并拉取项目

    # 一、mac上新建git并拉取项目

    # 1.1 安装git

    打开mac终端 输入git命令，未安装git的mac会提示是否使用命令行模式，选择同意并下载安装，如果下载途中出现未找到最新软件信息的报错问题，则进入网址https://developer.apple.com/download/more/下载最新的插件![image-20201021153256015](/Users/yxy/Library/Application Support/typora-user-images/image-20201021153256015.png)

    安装成功后再次下载git，git下载成功后，可以通过git --version检查是否成功，成功会显示![image-20201021153544119](/Users/yxy/Library/Application Support/typora-user-images/image-20201021153544119.png) 。

    # 1.2.1 对git进行全局配置

    git config --global user.email "you@example.com"  
git config --global user.name "Your Name"

    # 1.2.2 检查是否存在SSH KEYS

    cd ~/.ssh

    # 1.2.3 如果存在KEYS ，则备份

    $ls -a

$mkdir key_backup

$cp id_rsa* key_backup

$rm id_rsa* (看情况是否删除)

    # 1.2.4 创建新的KEY

    ssh-keygen -t rsa -C  "youremail"

    # 1.2.5 查看生成的KEY

    cat id_rsa.pub 把里面内容全部复制

    # 1.2.6 配置 GItHub

    在账户中找到 settings --> SSH and GPG key -->配置好即可

    # 1.3.1 git 拉取操作

    首先打开网页端的github (opens new window),找到对应项目，获取项目ssh 或 http地址

    ![image-20201021160244638](/Users/yxy/Library/Application Support/typora-user-images/image-20201021160244638.png)

    然后新建一个文件夹，在文件夹目录打开终端，执行

    git clone ssh地址

    ![image-20201021160634430](/Users/yxy/Library/Application Support/typora-user-images/image-20201021160634430.png)

    显示如上图则代表下拉成功
